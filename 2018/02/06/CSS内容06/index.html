quotes

设置嵌套引用（embedded quotation）的引号类型。
取值：
none，规定 “content” 属性的 “open-quote” 和 “close-quote” 的值不会产生任何引号。
string string string string ，定义要使用的引号。前两个值规定第一级引用嵌套，后两个值规定下一级引号嵌套。也可以有三级或者四级嵌套。
注意：

  • 设置的值只会对q标签引用的内容生效。

最后在页面上的显示效果为：[a<b”c”d>e]f。

  • 如果想对非q标签引用的内容生效，那么需要加上content属性。例子见content一节的内容。

    counter-reset

  • 创建或重置一个或多个计数器。
  • 取值：
    none：默认。不能对选择器的计数器进行重置。
    id number：id 定义将增加计数的选择器、id 或 class；number 定义增量。number 可以是正数、零或者负数。
  • 主要作用就是给计数器起个名字。如果可能，顺便告诉下从哪个数字开始计数。默认是0

你也可以创建多个计数器

多个计数器不用逗号，直接使用空格就好。

counter-increment

设置某个选取器每次出现的计数器增量。默认增量是 1。
取值：
none，阻止计数器增加。
id number，id 定义将增加计数的选择器、id 或 class；number 定义增量。number 可以是正数、零或者负数。
注意：如果使用了 “display: none”，则无法增加计数。如使用 “visibility: hidden”，则可增加计数。

content

用来和:after及:before伪元素一起使用，在对象前或后显示内容。
取值：

  • noraml：默认值，和none相同
  • none：不生成任何值

  • string：插入字符串

  • attr:插入标签属性值

  • url:使用指定的绝对或相对地址插入一个外部资源（图像，声频，视频或浏览器支持的其他任何资源）

  • counter(name):使用已命名的计数器

  • counter(name,list-style-type):使用已命名的计数器并遵从指定的list-style-type属性

高级用法：

  • counters(name,string):使用所有已命名的计数器
    couter(name,style-type)和counters(name,string)的用法实际上是一样的，只不过如果使用后者的话，第二个参数一定得是字符串。
  • counters(name,string,list-style-type)
    这种方法和counters(name,string)方法实现的效果完全可以由前面提到的高级用法同样实现，只不过该方法需要用到的代码量更少。具体例子参见counter-reset一节的内容。
  • open-quote：插入quotes属性的前标记
  • close-quote：插入quotes属性的后标记
